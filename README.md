# Anime Clash: Ultimate Arcade

Anime Clash: Ultimate Arcade is a hyper-fast 2D fighting game built entirely in a single HTML file! Brawl with 12 iconic anime legends across dynamic arenas using air-dashes, mid-air combos, and screen-shattering ultimates. With 100% procedural art, synth music, local multiplayer, and a ruthless AI, it's pure, hype-driven action!

## 🌟 Features

*   **Zero Dependencies:** The entire game logic, collision, styling, canvas drawing, and physics are self-contained in a single `index.html` file.
*   **Intelligent CPU AI:** The CPU opponent actively measures distances, blocks projectiles, and executes "cross-up" jump-attacks to break your guard if you play too defensively.
*   **Local Multiplayer:** Play head-to-head with a friend on the same keyboard.
*   **Custom Control Mapping:** A built-in UI allows both P1 and P2 to freely remap their controls. Conflict-detection automatically swaps keys so players never get stuck.
*   **Dynamic Audio System:** Procedurally generated 8-bit chip-tune hit sound effects and background music. Includes a global mute (`M`) toggle.
*   **Responsive Physics Speed:** High-gravity falling, variable jump-heights, and momentum-carrying dashes provide a razor-sharp, competitive game feel.

## ⚔️ Combat Mechanics

*   **Combos & Hit-Stun:** Land rapid attacks to lock the opponent in hit-stun.
*   **Dash & Cross-Ups:** Press Dash (`Shift` or `O`) to quickly close distance or escape. If an opponent is blocking, dash *through* or jump *over* them and hit them in the back to trigger a **Guard Break**. A broken guard forces the player to visually release and re-press the block key.
*   **Universal Beam Pushing:** Unleash your character's Ultimate ability! Every ultimate projectile physical catches the opponent, violently bends them backward, and violently shoves them across the arena until they collide with the map boundary for massive execute damage and screen-shake!
*   **Beam Clashes:** If two characters fire their ultimates at the exact same time, the projectiles will collide in the center of the screen, triggering a massive, cinematic 60-frame energy clash explosion.

## ⌨️ Default Controls

Keys can be fully customized in the "CONTROLS" menu!

| Action | Player 1 | Player 2 |
| :--- | :--- | :--- |
| **Move Left**  | `A` | `ArrowLeft` |
| **Move Right** | `D` | `ArrowRight` |
| **Jump**       | `W` | `ArrowUp` |
| **Attack**     | `F` | `K` |
| **Block**      | `G` | `L` |
| **Dash**       | `Shift` | `O` |
| **Ultimate**   | `U` | `I` |

*   **Global Audio Toggle:** `M` or the onscreen UI button.

## 🚀 How to Play

Simply open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge) and click anywhere to start! No servers or installation required.
